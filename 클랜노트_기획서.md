# 클랜 노트 (Clan Note) 기획서

> 서든어택 클랜을 위한 전술 커뮤니케이션 플랫폼  
> 서비스 URL: https://clan-note.vercel.app  
> 최종 업데이트: 2026-06-12

---

## 1. 서비스 개요

### 서비스명
**클랜 노트 (Clan Note)**

### 한 줄 소개
서든어택 클랜의 전술 설계, 팀원 관리, 일정 조율, AI 피드백을 한 곳에서 제공하는 웹 기반 플랫폼

### 배경 및 문제 인식
- 서든어택은 클랜 문화가 강한 게임이나, 클랜 전술 공유를 위한 공식 도구가 없음
- 넥슨은 2023년 1월 클랜 랭크전을 출시했으나 클랜 내 커뮤니케이션 도구는 미제공
- 디스코드 화면 공유 시 서든어택이 검은 화면만 표시됨 — 안티치트/보안 정책으로 화면 캡처 API가 차단된 것으로 추정 (DC인사이드, 넥슨 공식 커뮤니티 다수 신고)
- 팀원들이 맵을 보며 실시간으로 전술을 공유할 수단이 부재

## 2. 주요 기능 (현재 구현 완료)

### 2-1. 전술 맵보드 ✅
**핵심 기능**
- 서든어택 전 맵 지원
- Canvas API 기반 실시간 드로잉
  - 경로(펜) 도구 — 마우스 커서 펜 모양으로 변경
  - 화살표 도구
  - 직선 도구
  - 스티커 배치 (폭탄, 집결 등 전술 마커)
- 팀원 현황 패널 (캔버스 좌하단 오버레이)
  - 접속 상태 (온라인/오프라인) 표시
  - 드래그 앤 드롭으로 맵에 팀원 마커 배치
- Ctrl+Z 실행취소
- 작전 메모 텍스트 영역
- URL 공유 버튼 (클립보드 복사)
- 디스코드 음성 채널 입장 버튼

**레이아웃**
- 상단 GNB + Topbar (작전명, 맵 선택, 실행취소/초기화/저장)
- 좌측 52px 툴바 (드로잉 도구)
- 중앙 캔버스 (맵 이미지 + 드로잉 레이어)
- 우측 260px 패널 (스티커 선택, 색상, 굵기, 작전 메모)

### 2-2. 클랜 커뮤니티 ✅
- 공지사항, 작전 공유, 경기 결과 게시 공간 (프로토타입)

### 2-3. 클랜 캘린더 ✅
- 주간 가용 시간 그리드 (월~일, 18시~02시)
- 클랜원이 직접 게임 가능 시간 체크 (읽기/쓰기)
- 시간대별 참여 인원 색상 강도로 시각화
- 최적 스크림 시간대 자동 추천
- 주 단위 네비게이션

### 2-4. AI 전술 피드백 ✅
- 전장 리플레이 파일 업로드 (.json)
  - 파일 파싱 후 경기 스탯 자동 입력
- 경기 상황 텍스트 입력
- AI 피드백 생성 (포지셔닝, 타이밍, 팀 협력도)

### 2-5. 멤버 관리 ✅
- 클랜원 역할 설정 (마스터/부마스터/엘리트/일반)
- 활동 현황 확인
- 멤버 초대 (프로토타입)

## 5. 시장 검증 현황

### 확인된 페인포인트
- 디스코드 화면 공유 시 서든어택 검은 화면 표시 → 맵 보며 소통 불가 (보안 정책으로 화면 캡처 차단 추정, 다수 신고 확인)
- 넥슨 공식 클랜 랭크전 운영 중이나 클랜 내 커뮤니케이션 도구 미제공

### 미검증 가설
- 카카오톡 단톡방 + 손그림으로 전술 공유한다는 주장 → 공개 출처 미확인
- 실제 니즈 크기는 직접 유저 인터뷰로 검증 필요

### 확장 가능성
- 서든어택 한정이 아닌 FPS 클랜 범용 도구로 포지셔닝 시 타깃 확대 가능
- 맵 이미지 교체로 배틀그라운드, 발로란트 등 타 게임 지원 가능
